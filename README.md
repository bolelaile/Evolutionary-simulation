# 生态系统模拟器

生态系统模拟器是一个简单的Python项目，使用Matplotlib库来模拟草食动物（Herbivores）和捕食者（Predators）在一个二维环境中的动态互动。该模拟包括食物生成、生物移动、觅食、繁殖、体力管理等特性，并通过实时动画展示这些过程。
